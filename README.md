# turanly.com - Portfolio Website

Turan Hidayetov-un şəxsi portfolio saytı.

## Tech Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **next-themes** (dark/light mode)
- **Framer Motion** (animasiyalar)

## Dillər

- Azərbaycan (az) - default
- İngilis (en)
- Rus (ru)

## Development

```bash
npm run dev    # Development server
npm run build  # Production build
npm run start  # Start production server
```

## Deploy

Vercel-ə deploy etmək üçün GitHub repo-nu Vercel-ə bağlayın.
